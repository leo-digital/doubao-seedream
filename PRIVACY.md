# 隐私政策

本插件（DIFY平台“火山引擎Seedream”工具插件）非常重视您的隐私。本隐私政策旨在透明地向您说明我们如何处理您的数据。

## 1. 我们收集的数据类型

为了提供图像生成服务，本插件可能会处理以下数据：

- **用户输入信息**：您在使用插件工具时输入的文本提示词（Prompt）。
- **凭据信息**：您提供的火山引擎（Volcengine）Ark API Key。这些信息仅用于向火山引擎发起服务调用。
- **配置信息**：您提供的 API 基础地址（Base URL）以及生成参数（如图片尺寸、是否添加水印等）。

本插件**不直接收集**您的姓名、电子邮件、电话号码或地理位置等直接个人标识信息。

## 2. 数据的使用方式

我们使用处理的信息仅用于以下目的：

- **提供服务**：根据您的提示词和参数，调用火山引擎的 Seedream 模型生成图片。
- **身份验证**：使用您的 API Key 代表您向火山引擎发起请求。

## 3. 第三方服务与数据共享

为了实现图像生成功能，本插件会向以下第三方服务发送数据：

- **服务商**：火山引擎（Volcengine / 北京字节跳动科技有限公司）。
- **发送数据内容**：您的提示词、生成的配置参数以及 API Key。
- **目的**：执行图像生成算法。
- **第三方隐私政策**：请参考 [火山引擎隐私政策](https://www.volcengine.com/docs/6256/64902) 了解其数据处理实践。

## 4. 数据存储与安全

- **凭据安全**：您的 API Key 由 Dify 平台安全托管，插件代码仅在运行时通过环境变量或参数获取，不进行持久化存储。
- **数据留存**：本插件本身不存储您的任何输入文本或生成的图片。这些数据通常由 Dify 平台或您连接的火山引擎账户根据其各自的政策进行处理。

## 5. 常见问题 (FAQ)

**问：本插件是否会收集我的个人身份信息？**
答：不会。我们仅处理生成图片所必需的提示词和 API 凭据。

**问：我的数据会用于模型训练吗？**
答：本插件不使用您的数据进行训练。火山引擎是否使用数据进行训练取决于您与火山引擎之间的服务协议及其隐私政策。
