# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2026-03-06

### Added
- Support `imageUrls` parameter, allowing users to pass image URLs directly as concatenated strings for more flexible image referencing.

---

## [1.0.0] - 2026-03-02

### Added
- **Integrate Seedream 5.0-lite model** with efficient image generation capabilities.
- **Markdown output support**: Generate results can be output as Markdown text for better display.
- **Web search enhancement**: Clarified web search tool description for text-to-image effectiveness.

### Fixed & Improved
- Upgraded `volcengine-python-sdk` version.
- Enhanced logging for better debugging.
- Updated icon files and `manifest.yaml` configuration.
- Adjusted default watermark settings.

---

## [0.0.1] - 2026-01-14

### Added
- **Initial release**: Integrated Volcengine **Doubao Seedream V4.5** model with text-to-image and image-to-image generation capabilities.
